# 🎯 Goal Manager

#### Project Status: `👨‍💻 Closed alpha`

**Goal Manager** will help you achieve your goals:
1. Visualize goals.
2. Create sub-goals.
3. Monitor the progress of implementation.
4. Keep notes about Goal.
